# Proxy Checker & Scraper

A lightweight tool for scraping public proxies and checking their availability.

## Features

- Scrape proxies from multiple sources
- Check HTTP, HTTPS, SOCKS4, and SOCKS5 proxies
- Fast concurrent validation
- Filter out dead proxies
- Save working proxies
- Display response times and status
